В этой папке собрано все, что относится к расширениям HandyCache.
В файлах с расширением lua находятся сами расширения. Вы можете пользоваться ими
или использовать их в качестве примеров для написания своих расширений.
Чтобы начать использовать расширение, его нужно добавить в список
расширений. Для этого используйте кнопку Добавить расширение на вкладке
Настройки/Общие/Расширения.
В файле HCExtensions.html находится описание устройства расширения, а также
описание переменных и функций, предоставляемых HandyCache.
В файле LuaTest.exe находится программа для редактирования и тестирования
расширений.
