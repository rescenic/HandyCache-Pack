
	HandyCache Pack

-----------------------------------------------------------------------

	Caching proxy server

- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -

	Use:

	1. Install (from folder "Support\!!! Must install before first run HandyCache !!!\"):
		1a. Visual C++ Redistributable for Visual Studio 2012 Update 4
		1b. Certificates for browsers
	2. Run HandyCache.exe
	3. Setting your browser setting Proxy server to:
		ip: 127.0.0.1
		port: 3128
